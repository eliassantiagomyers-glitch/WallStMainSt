# Wall Street v. Main St Final Review

A minimalist, responsive flashcard application for studying final exam terms.

## Features
- **52 Key Terms**: Covers the complete list of identifications.
- **Minimalist Design**: Clean, distraction-free "typewriter" aesthetic.
- **Mobile Optimized**: Works great on phones and desktops.
- **Keyboard Support**: Space to flip, Arrows to navigate.

## How to Use
Simply open `index.html` in any web browser.

## Deployment
This project is ready for **GitHub Pages**. Simply enable Pages in your repository settings and point it to the `main` branch.
